# Art Inspiration

- **Direction:** natural wonder with digital zen UI
- **Aesthetic:** premium fitness with clean, high-contrast readability
- **Constraint:** visuals should serve the PC-first camera workout experience first, while staying portable to later mobile and VR work where possible
- **Nature Inspired:** choose environments inspired by strong outdoor or studio energy

## Art domains that still matter

- **Environments** for workout atmosphere
- **Avatars** for player/coach identity
- **Cosmetics** for controlled profile customization
- **Core internal assets** for UI, branding, and runtime presentation

## Scope note

Older docs emphasized package-local gameplay skins/assets as a creator-facing product pillar. The current docs slice shifts customization toward controlled avatar/cosmetics progression instead.
