# Athlete Preferences & Overrides

This document describes athlete-controlled visual and gameplay comfort settings.

## Philosophy

Athletes should be able to preserve comfort and readability without rewriting authored workout content.

## Preference categories

### Visual overrides

- **Preferred Environment**
- **Avatar Visibility**
- **Coach Visibility**

### Menu customization

- **Main Menu Background**

### Gameplay defaults

- **Preferred Mode:** Boxing or Flow
- **Preferred View Type:** Portal or Track
- **Preferred Difficulty**
- **Accessibility Modifiers**

## Scope note

Older docs described persistent skin overrides for workout-package gameplay assets. The current product direction is moving away from package-local gameplay asset swaps and toward controlled avatar/cosmetics customization.
