# Attribution Guide

If you fork AeroBeat or use our assets, you must provide proper credit. Here is how to do it correctly.

### 1. In Your Source Code

*   **Do Not Remove Headers:** You must keep the existing copyright headers in all source files (GPLv3/MPL 2.0 requirement).
*   **Add Your Own:** If you modify a file, add your own copyright line below ours.
    ```gdscript
    # Copyright (c) 2026 AeroBeat-Fitness
    # Copyright (c) 2027 Your Name/Company
    ```

### 2. In Your Game Client

If you release a build (even a non-commercial one), you must credit AeroBeat in the user interface.

*   **Credits Screen:** Include "Powered by AeroBeat" or "Based on AeroBeat Platform".
*   **Assets:** If you use our CC BY-NC 4.0 assets, list "AeroBeat-Fitness" in your art credits.
